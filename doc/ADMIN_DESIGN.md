# Визуальный дизайн админки (по референсу Dealstack)

Структура интерфейса админ-панели ИМ, ориентированная на тёмную тему и удобную навигацию.

---

## 1. Общая концепция

- **Тёмная тема:** тёмно-серый/чёрный фон, белый/светло-серый текст, акцентный цвет (фиолетовый/лайм).
- **Два столбца:** фиксированный левый сайдбар (навигация) + основная область контента справа.
- **Чёткая иерархия:** заголовки, отступы, группировка разделов.
- **Минимум визуального шума:** аккуратные карточки, чёткие границы, читаемые формы.

---

## 2. Структура экрана

### Левая панель (сайдбар)

| Элемент | Описание |
|--------|----------|
| **Брендинг** | Логотип и название «Hardcode Store» вверху. |
| **Основной раздел** | Обзор (Dashboard), Товары, Заказы, Клиенты. |
| **Справочники** | Категории, Бренды, Атрибуты, Способы доставки. |
| **Состояние** | Подсветка активного пункта (акцентный фон), иконки у пунктов меню. |
| **Бейджи** | При необходимости: счётчики (например, новых заказов) рядом с пунктами. |

### Правая область (контент)

| Элемент | Описание |
|--------|----------|
| **Шапка** | Поиск, breadcrumbs, уведомления. |
| **Заголовок страницы** | Название раздела (например, «Товары», «Заказ #123»). |
| **Действия** | Кнопки «Добавить», фильтры, экспорт и т.п. |
| **Основной контент** | Таблицы, карточки, формы. |
| **Данные в карточках** | Ключевые метрики в блоках (при необходимости). |

---

## 3. Структура навигации под наш ИМ

**ОСНОВНОЕ**
- Обзор (Dashboard) — главная админки, краткие метрики.
- Товары — список товаров, добавление/редактирование.
- Заказы — список заказов, просмотр, смена статуса.

**СПРАВОЧНИКИ**
- Категории — иерархия каталога.
- Бренды — справочник брендов.
- Атрибуты — цвет, размер и др.
- Способы доставки — ТК и варианты доставки.

**СИСТЕМА**
- Пользователи — при появлении кастомных пользователей.

## 4. Цветовая схема

| Назначение | Цвет | Использование |
|-----------|------|----------------|
| Фон сайдбара | #0f0f14, #1a1a24 | Основной тёмный фон. |
| Фон контента | #15151c, #1e1e28 | Немного светлее. |
| Акцент | #8b5cff, #a78bfa | Активные пункты, кнопки, выделения. |
| Текст | #f5f5f7, #e5e5e7 | Основной текст. |
| Приглушённый текст | #9ca3af, #6b7280 | Второстепенные надписи. |
| Границы | rgba(255,255,255,0.08) | Разделители и рамки. |
| Успех | #22c55e | Статусы, подтверждения. |
| Ошибка | #ef4444 | Ошибки, предупреждения. |

---

## 5. Типографика

- **Заголовки разделов:** 18–20px, полужирный.
- **Заголовки карточек:** 14–16px, полужирный.
- **Основной текст:** 14px, обычный.
- **Второстепенный текст:** 12–13px, серый.
- **Шрифт:** системный sans-serif или Inter/Roboto.

---

## 6. Компоненты

- **Таблицы:** тёмный фон, чёткие границы строк, hover-подсветка.
- **Формы:** контрастные поля, понятные лейблы, акцентные кнопки «Сохранить».
- **Карточки:** тёмный фон, тонкая рамка, внутренние отступы.
- **Бейджи статусов:** цвет по статусу (новый — синий, оплачен — зелёный и т.п.).
- **Кнопки:** акцентная основная кнопка, нейтральные вторичные.

---

## 7. Адаптивность

- На узких экранах сайдбар сворачивается в гамбургер-меню.
- Таблицы допускают горизонтальный скролл или адаптивный вид (карточки).
- Формы остаются читаемыми и удобными для ввода на планшетах и телефонах.
