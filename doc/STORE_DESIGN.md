# Дизайн страницы магазина — «Утро современный» стиль

Референсы: Nestify (десктоп), LuxiQue / косметика (мобайл). Стиль — светлый, воздушный, минималистичный.

---

## 1. Общая концепция

- **Стиль:** «Утро современный» — светлый фон, много воздуха, мягкие тени, скругления, glassmorphism (матовое стекло) для оверлеев.
- **Палитра:**
  - Фон: `#f8f9fc`, `#ffffff`
  - Текст: `#1a1d29`, `#4a4d5a`
  - Акцент: `#6366f1` (индиго), `#8b5cf7` (фиолетовый)
  - Вторичный: `#94a3b8`, `#cbd5e1`
  - Рейтинг: `#f59e0b`
- **Типографика:** Чистый sans-serif (Inter, SF Pro, system-ui). Заголовки — полужирные, крупные. Описания — серые, мельче.

---

## 2. Компоненты

### Десктоп

| Элемент | Описание |
|---------|----------|
| **Шапка** | Логотип + поиск (широкая строка) + иконки (избранное, корзина). Светлый фон, тонкая тень. |
| **Категории** | Горизонтальные «пилюли» — скролл по категориям (Table, Sofa, Chair и т.д.). |
| **Сетка товаров** | 2–3 колонки. Карточки с изображением, ценой, рейтингом, кнопками «в корзину» / «в избранное». |
| **Карточка товара** | Изображение, glassmorphism-оверлей с ценой/названием/рейтингом, иконки действий. |
| **Сайдбар (опционально)** | «Наша команда», «Подписка на рассылку» — блоки с воздухом. |

### Мобилка

| Элемент | Описание |
|---------|----------|
| **Шапка** | Стрелка назад, поиск, корзина (с бейджем кол-ва). |
| **Фильтры** | Чипы категорий (All, Jacket, Jeans…), горизонтальный скролл. |
| **Сетка** | 2 колонки, компактные карточки. |
| **Карточка товара** | Фото, название, цена, сердце. |
| **Страница товара** | Большое фото, бейджи (Vegan, Natural), размер, количество, «В корзину». |
| **Корзина** | Список позиций, промокод, итог, «Оформить заказ». |

---

## 3. Структура страниц

1. **Главная / каталог** — hero (опционально), поиск, категории, сетка товаров.
2. **Карточка товара** — галерея, название, цена, варианты (размер/цвет), количество, «В корзину».
3. **Корзина** — список, промокод, итог, кнопка оформления (логика оформления — позже, без авторизации пока).

---

## 4. Анимации

- Плавное появление карточек при скролле (fade-in, лёгкий slide-up).
- Hover на карточках: лёгкое поднятие (transform + тень).
- Переходы между страницами: мягкие (opacity).
- Кнопки: лёгкий scale при нажатии.

---

## 5. Адаптивность

- **Desktop:** max-width ~1200px, 3 колонки.
- **Tablet:** 2 колонки, сжатая шапка.
- **Mobile:** 2 колонки, бургер-меню при необходимости, sticky-корзина/кнопка.

---

## 6. Без авторизации (MVP)

- Корзина в `localStorage` или session.
- Нет избранного (или локально).
- Оформление заказа — форма (имя, телефон, email, адрес) без регистрации.
