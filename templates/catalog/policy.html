Политика конфиденциальности

Hardcode Store — интернет-магазин. Мы собираем личные данные только для оформления и доставки заказов.

Какие данные собираем

  • Имя — для обращения при доставке
  • Email — для подтверждения заказа и уведомлений
  • Телефон — для связи курьера или СДЭК при доставке
  • Адрес доставки — город, улица, дом, квартира — только для доставки товара

Цель сбора

Все данные используются исключительно для:

  • обработки и выполнения заказа
  • связи с покупателем по вопросам заказа
  • доставки товара по указанному адресу

Безопасность

Сайт работает по защищённому протоколу HTTPS. Личные данные не передаются третьим лицам, кроме служб доставки (курьер, СДЭК) для выполнения заказа.
